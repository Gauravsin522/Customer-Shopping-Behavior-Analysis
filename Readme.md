# Customer Shopping Behavior Analysis

![Dashboard](images/dashboard.png)

## Project Overview
This repository provides an end-to-end analysis of customer demographics, purchase patterns, category spending, and behavioral trends. The objective is to convert raw transactional data into actionable business intelligence.

## Key Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Category‑wise revenue insights
- Subscription behavior analytics
- Customer segmentation
- Power BI interactive dashboard

## Dashboard
The following image represents the full analytics dashboard summarizing customer behavior:

![Customer Behavior Dashboard](images/dashboard.png)

## Dataset Details
- Customer Demographics (Age Group, Gender)
- Purchase Amount
- Category & Product Preferences
- Shipping Type
- Subscription Status
- Review Ratings

## Tech Stack
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Power BI
- Jupyter Notebook

## Insights
- Young Adults contribute highest revenue.
- Clothing category dominates sales volume.
- Majority are non‑subscribers—loyalty program potential.
- Digital & express shipping preferred by high‑value customers.

## Recommendations
- Expand loyalty program offerings.
- Optimize inventory for high‑demand categories.
- Push express/digital‑first shipping incentives.
- Use satisfaction clusters for targeted campaigns.
